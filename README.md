
## [To Do List][1] is project created when studying with [JetBrains Academy Hyperskill course][2]

#### **To_Do_List** is using SQLAlchemy it allows the user to:
* Store tasks
* Add and delete tasks
* Show: 
  * Missed tasks according to current date
  * All tasks
  * Week's tasks
  * Today's tasks
